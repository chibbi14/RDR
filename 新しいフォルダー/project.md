## ATC Simulator v92.8 設計仕様・要件定義書

### 1. プロジェクト概要
米子空港（RJOH）および出雲空港（RJOC）周辺の空域を監視・制御するReactベースの航空管制（ATC）レーダーシミュレーター。高精度な座標計算と航空図に基づいたプロシージャの可視化を目的とする。

### 2. システム構成・技術スタック
- **Framework:** React (Functional Components, Hooks)
- **Styling:** Tailwind CSS
- **Icons:** Lucide React
- **Coordinate System:** 地理的緯度経度からピクセルへの動的変換
- **State Management:** レイヤー別・空港別プロシージャ表示制御

---

### 3. 数学的基盤 (不変ロジック)

#### 3.1 座標変換パラメータ
- **投影基準点 (RJOH ARP):** Lat 35.493333, Lon 133.239167
- **基準スケール:** 1.0 NM = 10px (Zoom=1.0時)
- **経度補正係数:** `cos(35.493333 * PI / 180)` (約 0.814)

#### 3.2 基本計算式
**緯度経度から相対座標(NM)への変換:**
- `y_nm = (lat - ARP.lat) * 60`
- `x_nm = (lon - ARP.lon) * 60 * cos(ARP.lat)`

**ピクセル座標変換 (toNMPos):**
- `y = -(lat - ARP.lat) * 60 * 10 * zoom`
- `x = (lon - ARP.lon) * 60 * 補正係数 * 10 * zoom`

**Dead Reckoning (calcRelativeLatLon):**
`newLat = lat + (dist * cos(brg)) / 60`
`newLon = lon + (dist * sin(brg)) / (60 * cos(lat))`

**DMS 座標変換 (dmsToDeg):**
`Degree + Minute/60 + Second/3600`

**交点計算 (getIntercept):**
2つの点とそれぞれの方位（Heading）から交点座標を算出する幾何計算ロジック。

### 5. 空域構造 (ACA/MVA)

#### 5.1 ACA (Airspace Control Area)
米子進域管制区の境界。相対座標系(NM)における12個の制御点（`acaPoints`）を、直線および特定の円弧（R-34NM, R-25NM, R-15NM）で接続して描画する。

#### 5.2 MVA (Minimum Vectoring Altitude)
米子空港を中心に構成される高度制限グリッド。
- **Radials:** 020, 035, 050, 087, 100, 110, 145, 210, 242, 268, 335, 360 度等。
- **Arcs:** 3, 7, 9, 10, 13, 15, 18, 25, 30, 35, 40, 50 NM。
- **Altitude Labels:** 15 (1500ft) ～ 160 (16000ft)。

#### 5.3 描画実装仕様 (SVG)
**ACA (ポリゴン描画):**
- 属性: `fill="none"`, `stroke="yellow"`, `strokeWidth="0.7"`, `strokeDasharray="5,3"`, `opacity="0.4"`

**MVA (セクタ描画):**
- 境界属性: `fill="none"`, `stroke="#00ffff"`, `strokeWidth="0.4"`, `opacity="0.2"`
- ラベル属性: `fill="#00ffff"`, `fontSize="7"`, `opacity="0.4"`, 表示位置はセクタの第1頂点オフセット

### 6. プロシージャ描画定義 (Master Routes)
すべての経路は、直線(line)または円弧(arc)の配列で構成される。

#### 6.1 RJOH 主要経路
- **SID:** INABA, SOUTH, DOZEN, USAGI, STAGE (RWY07/25両対応)。
- **TRSN:** MS (Matsue), TZ (Tozan), MZ (Maizuru) への遷移経路。
- **STAR:** GAINA (EAST/WEST), KYURI (EAST/WEST)。
- **IAP:** ILS Z/X RWY25, RNP Z/Y RWY07, TACAN A。

#### 6.2 RJOC 主要経路 (v92.8 強化項目)
- **SID:** DOZEN, MATSUE, TAKHI, SAIGO (RWY07/25両対応)。
- **STAR:** SUSAR, OKUNI, NAKAU STAR。
- **IAP:** RNP RWY07 (via SUSAR/OKUNI), RNP RWY25。

### 7. 開発ガイドライン
- **データロック:** セクション4, 5, 6の数値はプログラムの「コア」であり、UIの変更等でこれらを書き換えてはならない。
- **描画レイヤー:** 下層から順に `ACA` -> `MVA` -> `Procedures` -> `Fixes` -> `Airport Symbols`。
- **インタラクション:** Zoom変更時は `nmToPx` 関数を介してすべての描画要素を再計算する。
